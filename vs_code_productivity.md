# VS Code Productivity – Reflection

## Useful VS Code Shortcuts

### Navigation

- Ctrl + P → Quick file search  
- Ctrl + Shift + O → Go to symbol in file  
- Ctrl + Tab → Switch between open files  

### Editing

- Alt + Click → Multiple cursors  
- Ctrl + D → Select next occurrence  
- Ctrl + / → Comment or uncomment code  
- Shift + Alt + ↓ → Duplicate line  

### Debugging

- F5 → Start debugging  
- F9 → Toggle breakpoint  

### Terminal

- Ctrl + ` → Open/close terminal  
- Ctrl + Shift + ` → Create new terminal  

# Reflection

## Most Useful Shortcuts

The most useful shortcuts for my daily work are:
- Ctrl + P for quickly opening files  
- Alt + Click for using multiple cursors  
- Ctrl + D for editing repeated code  
- Ctrl + ` for accessing the terminal  

These help me save time and avoid repetitive actions.

## Improving Productivity

Keyboard shortcuts improve productivity by reducing the need to switch between keyboard and mouse. This makes coding faster and more efficient, especially when working on large projects.


## Key Learning

A key learning for me is that mastering shortcuts can significantly improve speed and workflow, especially when working on coding and data-related tasks.
