1. Which AI tools did you try?

- I have used GitHub Copilot as an extension in Visual Studio Code, which helped me generate code efficiently using simple and rough prompts.
- In addition, I used ChatGPT to understand and explain the generated code. This combination was particularly helpful, as it provided deeper insights into the logic and improved my overall coding knowledge.

2. What worked well? What didn’t?

# Worked well

- The ability to generate code quickly using simple prompts was highly effective.
- It reduced the time required to write code and made programming more approachable, even for individuals from a non-IT background.
- It also boosted my confidence and encouraged me to explore coding further.

# Didnt

- The generated code often lacked detailed explanations or comments.
- This made it challenging to fully understand the logic, especially for beginners who rely on step-by-step clarity.

3. When do you think AI is most useful for coding?

- AI is most useful when working under tight deadlines or when encountering complex issues that are difficult to resolve independently.
- In such situations, AI can assist in quickly generating solutions and guiding the problem-solving process, helping to maintain productivity and meet timelines effectively.
