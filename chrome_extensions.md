# Browser Extensions Setup – Reflection

## Installed Extensions

I installed the following browser extensions to support development and debugging:

- React Developer Tools  
  Used to inspect React component structure and understand how components are rendered.

- Redux DevTools  
  Helps in debugging application state and tracking how data changes over time.

- JSON Viewer  
  Makes API responses easier to read by formatting JSON data in a structured way.

- Lighthouse  
  Used to analyze performance, accessibility, and best practices of web applications.

## Why I Chose These Extensions

These extensions are useful for modern web development because they help in debugging, analyzing performance, and understanding how applications behave in real-time. They provide better visibility into frontend applications and improve development efficiency.

## Most Useful Learning

The most useful thing I learned was how tools like React Developer Tools and Redux DevTools allow deeper inspection of application behavior. It becomes easier to understand component structure and state changes, which helps in debugging issues more effectively.

## Reflection

Using browser extensions can significantly improve productivity by providing better insights into application behavior, performance, and data flow. These tools make debugging faster and help developers build more efficient and reliable applications.
